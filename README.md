# CoreAPI

CoreAPI is a simple networking framework.

## Installation

CoreAPI is available on [CocoaPods](http://cocoapods.org). Just add the following to your project Podfile:

```ruby
pod 'CoreAPI'
```

## Usage

Use it as follows:

```objective-c
#import <CoreAPI/CoreAPI.h>
 ```
